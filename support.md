# DexFuel Support

DexFuel is a local-first nutrition, macros, recipes, and lifting coach.

## Help

For support, open an issue at https://github.com/GucciGross/DexFuel/issues.

When reporting a problem, do not include medical records, medication information, meal photos, API keys, or other sensitive data. Include the app version, iPhone model, and a short description of what happened.

## Data And Privacy Requests

Core DexFuel data can be removed on-device from Profile > Clear data. For questions about an online AI request or production service records, open a support issue without posting sensitive request content. A private follow-up channel will be arranged if identity verification or sensitive details are necessary.

## App Review Notes

- No account or login is required.
- The app stores core profile and log data locally on the device.
- Dex AI and photo macro estimates are optional features.
- Peptide tracking is schedule support only and does not provide dosing, prescribing, or treatment advice.
- Rescue Mode private notes are used to build the displayed plan and are not saved as raw text.
